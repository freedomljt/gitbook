Scrollbar style

定义滚动条主题(代码中是仿Mac浏览器的滚动条)

::-webkit-scrollbar-track
{
  width: 15px;
  background-color: #fafafa;
  border: 1px solid #e9e8e9;
}

::-webkit-scrollbar-thumb
{
  width: 8px;
  background-color: #c7c7c7;
}
::-webkit-scrollbar-thumb:hover {
  background: #878787;
}

# 基础语法

## 基础语法

### 标题

```text
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 一级标题

### 二级标题

#### 三级标题

**四级标题**

### 字体

```text
**粗体**
*斜体*
***斜体加粗***
~~删除线~~
```

**粗体** _斜体_ _**斜体加粗**_ ~~删除线~~

### 引用

```text
>引用
>>引用
>>>>>>>引用
```

> 引用
>
> > 引用
> >
> > > > > > > 引用

### 分割线

```text
---
----
***
*****
```
